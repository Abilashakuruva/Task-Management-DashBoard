
# Task Management Dashboard

# Description:
A role-based task management application built with React that allows admins to create, assign, and manage tasks. Tasks are persisted using localStorage.

# Features:
Admin and User login system

Task creation with optional user assignment

Drag-and-drop task reassignment using react-dnd

Tasks stored in localStorage for persistence

View tasks by assigned users or unassigned

# Tech Stack:

Frontend: React

Drag & Drop: react-dnd

Storage: localStorage

# How to Run:

Clone the repository:

git clone <your-repo-link>


# Install dependencies:

npm install


# Start the development server:

npm run dev
